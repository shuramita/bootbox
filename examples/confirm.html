Confirm

bootbox.confirm can be used in one of two ways, as shown below. Using an options object offers the most customization, using the available options.

  • Basic usage, with callback

    bootbox.confirm("This is a standard confirm with a callback! Does it make sense?", function(result){ 
    console.log('This was logged in the callback!'); 
});

  • Basic usage, using options object. When using an options object, the only required options are message and callback. 

    bootbox.confirm({
    message: "This is a confirm, using an options object. Does it seem straight-forward to use?",
    callback: function (result) {
        console.log('This was logged in the callback!');
    }
})

Advanced Usage

The examples below demonstrate a variety of the available options.

  • Also applies to: Alert, Prompt, Custom With alternate button text and color 

    bootbox.confirm({
    message: "This is a confirm with custom button text and color! Do you like it?",
    buttons: {
        confirm: {
            label: 'Yes',
            className: 'btn-success'
        },
        cancel: {
            label: 'No',
            className: 'btn-danger'
        }
    },
    callback: function (result) {
        console.log('This was logged in the callback: ' + result);
    }
});

  • Also applies to: Alert, Prompt, Custom With icon and button text 

    bootbox.confirm({
    title: "Destroy planet?",
    message: "Do you want to activate the Deathstar now? This cannot be undone.",
    buttons: {
        cancel: {
            label: '<i class="fa fa-times"></i> Cancel'
        },
        confirm: {
            label: '<i class="fa fa-check"></i> Confirm'
        }
    },
    callback: function (result) {
        console.log('This was logged in the callback: ' + result);
    }
});

  • Also applies to: Alert, Prompt, Custom Demoing all locales 

    var locale = $('#locales').val();

bootbox.confirm({
    message: "This confirm uses the selected locale. Were the labels what you expected?",
    locale: locale,
    callback: function (result) {
        Example.show('This was logged in the callback: ' + result);
    }
});
