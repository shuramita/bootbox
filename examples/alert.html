Alert

bootbox.alert can be used in one of three ways, as shown below. Using an options object offers the most customization, using the available options.

  • Basic usage

    bootbox.alert("This is the default alert!");

  • Basic usage, with callback

    bootbox.alert("This is an alert with a callback!", function(){ console.log('This was logged in the callback!'); });

  • Basic usage, using options object

    bootbox.alert({
    message: "This is an alert with a callback!",
    callback: function () {
        console.log('This was logged in the callback!');
    }
})

    When using an options object, the only required option is message.

Advanced Usage

The examples below demonstrate a variety of the available options.

Custom Classes

  • Also applies to: Confirm, Prompt, Custom Use a custom CSS class to enable an alternate dialog transition (using Animate.css) 

    bootbox.alert({
    message: "This is an alert with additional classes!",
    className: 'rubberBand animated'
});

Dismiss On Overlay Click

  • Also applies to: Confirm, Prompt, Custom Set the backdrop option to true to allow your users to dismiss the alert by clicking anywhere on the overlay. 

    bootbox.alert({
    message: "This alert can be dismissed by clicking on the background!",
    backdrop: true
});

Using Locales

  • Also applies to: Confirm, Prompt, Custom Starting with Bootbox 5, you can change the localization settings on your dialogs on a per-dialog basis. This example demonstrates using the ar (Arabic) locale. 

    bootbox.alert({
    message: "This alert uses the Arabic locale!",
    locale: 'ar'
});

Sizes

Starting with Bootstrap 3.1, you can use two modifiers to change the size of your dialogs from the default. We use the values 'small' and 'large' to represent Bootstrap's 'modal-sm' and 'modal-lg', respectively.

  • Also applies to: Confirm, Prompt, Custom Small dialog 

    bootbox.alert({
    message: "This is the small alert!",
    size: 'small'
});

    Requires Bootstrap 3.1.0 or newer.

  • Also applies to: Confirm, Prompt, Custom Large dialog 

    bootbox.alert({
    message: "This is the large alert!",
    size: 'large'
});

    Requires Bootstrap 3.1.0 or newer.
